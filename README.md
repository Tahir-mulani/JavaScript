## 🧠 Topics Covered

### 1️⃣ Basics

* Variables (`let`, `const`, `var`)
* Data Types
* Operators
* Conditional Statements (`if-else`, `switch`)
* Loops (`for`, `while`, `do-while`)

---

### 2️⃣ Functions

* Function declaration
* Function expression
* Arrow functions
* Parameters & return values
* Callback functions

---

### 3️⃣ Arrays

* Array creation & traversal
* Important methods:

  * `push()`, `pop()`
  * `shift()`, `unshift()`
  * `map()`, `filter()`, `reduce()`
  * `forEach()`, `find()`, `includes()`

---

### 4️⃣ Strings

* String methods:

  * `slice()`, `substring()`
  * `indexOf()`, `lastIndexOf()`
  * `replace()`, `split()`
  * `toUpperCase()`, `toLowerCase()`
* Pattern-based problems

---

### 5️⃣ Objects

* Object creation
* Access & update properties
* Nested objects
* Object methods

---

### 6️⃣ Recursion

* Basics of recursion
* Stack understanding
* Classic problems:

  * Factorial
  * Fibonacci
  * Backtracking intro

---

## 🧩 Practice Problems

### 🟢 Easy

* Reverse a string
* Find maximum in array
* Count vowels
* Check palindrome

### 🟡 Medium

* Two Sum problem
* Longest substring without repeating characters
* Group anagrams
* Rotate array

### 🔴 Hard

* Sliding window problems
* Dynamic programming basics
* Recursion + backtracking problems
